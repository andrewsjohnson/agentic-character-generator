## Skill Check Formula

```
Skill check modifier = ability modifier + (Proficiency Bonus if proficient)
```

If not proficient, the modifier is just the relevant ability modifier.

## Proficiency Sources

- **Background** → 2 skill proficiencies + 1 tool proficiency
- **Class** → additional skill and armor/weapon proficiencies (see class description)

When multiclassing, gaining a level in a new class grants only **some** of that class's starting proficiencies.

## Proficiency Bonus (Multiclass)

Proficiency Bonus is based on **total character level**, not level in any single class.

## Saving Throws

```
Saving throw modifier = ability modifier + (Proficiency Bonus if proficient)
```

Each class grants proficiency in **two** saving throws (specified in the class description).

## Passive Perception

```
Passive Perception = 10 + Wisdom (Perception) check modifier
```

Include Proficiency Bonus if proficient in Perception, plus any other applicable modifiers.

**Example:** Wisdom 15 (+2 mod) with Perception proficiency (+2) → Passive Perception = 14.

## Notes for Agents

- The full skill list and skill-to-ability mapping (e.g., Athletics → Strength, Stealth → Dexterity) is in the "Playing the Game" chapter, not included in this SRD excerpt.
- When Proficiency Bonus increases at a new level, update **all** numbers that include it: skills, saving throws, attack rolls, spell save DC, spell attack bonus.
